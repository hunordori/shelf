Importing Books

To import books, click on the "Import Books" button in the navigation bar or use this link: Import Books.

Reordering Books

Make sure you have the correct books imported

  1. Go to the Reorder page
  2. Scan the barcodes*
  3. When you are done, click the "Submit" button
  4. On the result page, you will see the correct order of the books

*It helps if you set your scanner's "end charachter" to return. Usually it is the defualt.

Using the "Result" page

The table

The table displays the correct order of the book and the position changes.

Minimal Moves

Calculates the least amount of moves required to reorder the books. Use the checkbox to mark the step completed.
